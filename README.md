# 📊 GlobalTech Solutions - Customer Management System

This project is a Python-based Customer Management System developed for Module 8. It demonstrates the use of dictionaries, loops, functions, and basic data aggregation techniques to manage customer and project data.

---

## 🚀 Features

- Store and manage service categories with hourly rates  
- Maintain customer records using dictionaries  
- Perform customer lookups and updates  
- Track projects and calculate project costs  
- Analyze service usage and financial data  
- Generate customer summary reports  
- Use dictionary comprehensions for efficient data processing  
- Validate customer data with functions  
- Provide service recommendations based on past usage  

---

## 🧠 Concepts Used

- Python Dictionaries  
- Nested Data Structures  
- Loops and Conditionals  
- Functions  
- Dictionary Methods (`.keys()`, `.values()`, `.get()`)  
- List and Dictionary Comprehensions  
- Data Aggregation and Analysis
